# set option for searching 
```
# ignore case
:set ic
# /ignore\c -> ignore case for just one search command

# highlight all matching phrases
:set hls

# show partial matches for a search phrase(incsearch)
:set is

# disable
:set noic
:set nohlsearch

# temporarly disable
:nohls 

# at once
:set hls ic
```
